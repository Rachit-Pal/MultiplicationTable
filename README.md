# MultiplicationTable

### Multiplication Table wiith User Input!

```
# include<iostream>
using namespace std;

int main()
{
    int a;
    cout << "Enter the Number : ";
    cin >> a;

    for(int i = 1; i <= 10; i++)
    {
        cout<< a << " x " << i << " = " << i*a <<endl;
    }
    
    return 0;
}
```
